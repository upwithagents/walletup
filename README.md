# WalletUp

An agentic assistant for personal budgeting and financial decisions:
categorizing transactions, evaluating spending against a plan, and
flagging issues that don't fit neatly into a banking app's UI.

<br clear="left"/>

Not tied to any single provider. First integration is [Wallet by
BudgetBakers](https://budgetbakers.com/) (bank sync, categorization,
budgets) as a data source, not the product itself.

## Status

Early — actively being planned and restructured. No automated writes to
any connected provider yet; changes always go through human review first.

## Privacy

Anything disclosing transaction-level data stays out of version control.
