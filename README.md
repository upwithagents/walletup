# WalletUp

An agentic assistant for personal budgeting and financial decisions —
categorizing transactions, evaluating spending against a plan, flagging
issues, and helping with the decisions that don't fit neatly into a banking
app's UI.

WalletUp isn't tied to any single finance provider. The first integration is
[Wallet by BudgetBakers](https://budgetbakers.com/), used today for bank
sync, categorization, and budgets — but the goal is a system that helps with
budgeting and financial decision-making generally, with Wallet (or any other
provider) as a data source rather than the product itself.

## Status

Early — actively being planned and restructured. No automated writes to any
connected financial provider yet; changes always go through human review
first.

## Privacy

Anything disclosing transaction-level data (payees, amounts, account
details) stays out of version control.
